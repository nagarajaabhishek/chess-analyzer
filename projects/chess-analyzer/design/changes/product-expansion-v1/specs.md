# OpenSpec Specifications: Product Expansion v1

Refer to the user requirements for Package 1 through 4:
- Package 1: Board-state commands: clean speech and match on custom intents without mutating game state. Response sent back via redirect msg parameter.
- Package 2: Metrics logging: log calls, track status, and aggregate call rate, duration, D7 retention, game count, and hangup causes.
- Package 3: Robustness: Twilio hints. If fallback piece+target parser matches exactly 1 move, run confirmation loop before playing.
- Package 4: Website: standard aria/nav/main HTML elements, focus outlines, 4 crawlable static pages under static/pages/, and SEO sitemaps.
